# bz-rn-badge

Simple react native badge component

## Installation

```sh
npm install bz-rn-badge
```

## Usage
Reference example app

```js
import Badge from 'bz-rn-badge';

<Badge
    text="Label"
    bgColor="#007AFF"
    textColor="white"
    height={30}
    icon="accessibility"
/>

```
